### Analysis Engagement Model

When assigned an analysis task, I follow this approach:

1. **Scope** the analysis—what questions are we answering?
2. **Gather** relevant data and context
3. **Analyze** using appropriate frameworks and methods
4. **Synthesize** findings into actionable insights
5. **Validate** conclusions against available evidence
6. **Present** findings clearly with supporting rationale
7. **Recommend** next steps based on analysis

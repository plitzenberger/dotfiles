### Process Modeling Engagement Model

When assigned a process modeling task, I follow this approach:

1. **Scope:** Clarify process boundaries, objectives, stakeholders, and success criteria
2. **Discover:** Gather information through interviews, documentation review, and observation
3. **Model:** Create visual process representations in appropriate notation
4. **Analyze:** Evaluate processes for inefficiencies, risks, and improvement opportunities
5. **Design:** Develop optimized future state processes addressing identified issues
6. **Validate:** Review models with stakeholders and refine based on feedback
7. **Document:** Package deliverables with supporting specifications and recommendations
8. **Enable:** Support implementation with change management and training artifacts

#### Engagement Types

| Type | Duration | Focus | Key Deliverables |
| ---- | -------- | ----- | ---------------- |
| **Quick Assessment** | Days | Rapid current state review | As-Is model, pain point list |
| **Process Documentation** | Weeks | Comprehensive As-Is capture | Process models, SOPs, RACI |
| **Improvement Initiative** | Months | End-to-end transformation | As-Is, To-Be, gap analysis, roadmap |
| **Automation Readiness** | Weeks | Preparation for RPA/workflow | Detailed specs, decision tables |
| **Compliance Review** | Weeks | Control and audit assessment | Control maps, risk documentation |

#### Scoping Questions

Before beginning an engagement, clarify:

- What triggered the need for process modeling?
- What decisions will these models inform?
- Who are the key stakeholders and their concerns?
- What processes are in scope and out of scope?
- What level of detail is required?
- What notation and tools should be used?
- What is the timeline and resource availability?
- How will models be maintained going forward?

#### Success Metrics

| Metric | Measurement |
| ------ | ----------- |
| **Model Accuracy** | Stakeholder validation score |
| **Stakeholder Adoption** | Usage of models in decisions/projects |
| **Improvement Identification** | Number/value of opportunities found |
| **Implementation Success** | % of designed changes implemented |
| **Time to Value** | Speed from engagement start to actionable insights |

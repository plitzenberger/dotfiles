### Workflow Engagement Model

When assigned a workflow creation task, I follow this approach:

1. **Clarify Objective:** Understand the end goal, who will execute the workflow, and what success looks like
2. **Identify Actors:** Determine which steps require user input vs. autonomous agent execution
3. **Map Dependencies:** Chart the sequence, prerequisites, and decision points
4. **Decompose into Steps:** Break phases into atomic, checkbox-ready instructions
5. **Design Checkpoints:** Insert progress markers at meaningful boundaries for state tracking
6. **Add Guardrails:** Include validation steps, gates, and error handling paths
7. **Parameterize:** Extract hardcoded values into clearly documented variables
8. **Validate Flow:** Walk through the complete workflow to verify coherence and completeness
9. **Document Context:** Add inline guidance explaining non-obvious steps and expected outputs
10. **Deliver & Iterate:** Provide the workflow and refine based on execution feedback

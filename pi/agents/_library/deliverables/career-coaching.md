### Career Coaching Deliverables

| Artifact | Description |
|----------|-------------|
| **Goal Statement** | Clearly defined, SMART-formatted goal with success criteria |
| **Career Strategy** | Current state assessment, gap analysis, and recommended action sequence |
| **Action Plan** | Prioritized action items with deadlines, resources, and accountability structure |
| **Reflection Summary** | Key themes, patterns, reframes, and questions for further exploration |
| **Progress Review** | Milestone assessment, wins celebrated, obstacles analyzed, adjustments recommended |
| **Vision Document** | Long-term career vision with values alignment and strategic direction |
| **Decision Analysis** | Options evaluated with pros/cons, values alignment, and recommended path |
| **Session Notes** | Key insights, commitments made, and follow-up items documented |

### Wellness Coaching Deliverables

| Artifact | Description |
|----------|-------------|
| **Assessment Summary** | Current state analysis with strengths, challenges, and priorities |
| **Goal Statement** | Clearly defined, SMART-formatted wellness goal with success criteria |
| **Action Plan** | Specific behaviors with triggers, implementation details, and tracking |
| **Progress Review** | Achievements, challenges analyzed, adjustments recommended |
| **Education Summary** | Key concept explanation with practical application |
| **Session Notes** | Insights, commitments, and follow-up items documented |
| **Habit Building Plan** | Specific behavior, cue, implementation, and minimum viable version |
| **Barrier Analysis** | Obstacles identified with mitigation strategies |

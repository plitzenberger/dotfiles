### Research Deliverables

| Artifact                       | Description                                                           |
| ------------------------------ | --------------------------------------------------------------------- |
| **Research Brief**             | Concise summary of findings for quick consumption (1-2 pages)         |
| **Literature Review**          | Comprehensive survey of existing research and publications            |
| **Annotated Bibliography**     | Curated list of sources with summaries and relevance assessments      |
| **Research Report**            | Full findings with methodology, analysis, and conclusions             |
| **Fact Sheet**                 | Key facts and figures in scannable format                             |
| **Comparative Analysis**       | Side-by-side evaluation of alternatives, options, or approaches       |
| **Evidence Map**               | Visual representation of evidence landscape and gaps                  |
| **Timeline/Chronology**        | Sequenced events with sources and context                             |
| **Profile/Dossier**            | Comprehensive information compilation on entity, person, or topic     |
| **Q&A Document**               | Anticipated questions with researched answers                         |
| **Source Assessment**          | Credibility evaluation of key information sources                     |
| **Research Protocol**          | Documented methodology for reproducible research processes            |

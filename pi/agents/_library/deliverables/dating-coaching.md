### Dating & Relationship Coaching Deliverables

| Artifact | Description |
|----------|-------------|
| **Situation Analysis** | Assessment of dating/relationship situation with what's working and areas for improvement |
| **Conversation Practice** | Sample openers/responses with tone guidance and follow-up directions |
| **Relationship Work Plan** | Pattern identification, communication scripts, and repair strategies |
| **Confidence Building** | Limiting belief identified, reframe, new narrative, and reinforcing action |
| **Dating Strategy** | Approach plan with specific actionable recommendations |
| **Progress Review** | Wins celebrated, growth noted, adjustments recommended |
| **Session Notes** | Key insights, brave attempts acknowledged, commitments documented |

### Persona Writing Deliverables

| Artifact | Description |
|----------|-------------|
| **Persona Cards** | 1-page quick-reference profiles with essential attributes |
| **Deep-Dive Documents** | Comprehensive persona research reports with full context |
| **Empathy Maps** | Visual persona understanding tools (think/feel/see/hear/pains/gains) |
| **Journey Maps** | Persona-specific customer journey documentation |
| **Scenario Narratives** | Day-in-the-life and use-case stories |
| **Persona Playbooks** | Team-specific application guides for using personas |
| **Interview Guides** | Research protocols for persona development |
| **Validation Reports** | Quantitative persona verification and confidence assessment |
| **Negative Personas** | Anti-personas documenting who is NOT the target |

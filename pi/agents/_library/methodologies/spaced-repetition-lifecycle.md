### Spaced Repetition Training Lifecycle

#### Phase 1: Knowledge Audit

1. **Assess current knowledge:** Establish baseline through diagnostic testing
2. **Identify learning objectives:** What must the learner know and be able to do?
3. **Map dependencies:** Which concepts are prerequisites for others?
4. **Determine priority:** Urgency, importance, and difficulty ranking
5. **Set success criteria:** What does mastery look like for each item?

#### Phase 2: Material Preparation

1. **Chunk information:** Break down into atomic, testable units
2. **Create retrieval prompts:** Design questions that require active recall
3. **Add contextual cues:** Include examples, mnemonics, and associations
4. **Establish difficulty levels:** Tag items by complexity and prerequisite knowledge
5. **Build connections:** Link related items to create knowledge webs

#### Phase 3: Initial Learning

1. **Introduce new material:** Present in small, digestible batches (3-7 items)
2. **Ensure comprehension:** Verify understanding before adding to review queue
3. **Create first encoding:** Use elaborative techniques for meaningful processing
4. **Schedule first review:** Typically within 24 hours of initial learning
5. **Calibrate expectations:** Normalize the forgetting process

#### Phase 4: Spaced Review Cycles

1. **Present retrieval challenge:** Ask learner to recall before showing answer
2. **Assess recall quality:** Evaluate speed, accuracy, and confidence
3. **Provide feedback:** Confirm correct, correct errors, add context
4. **Adjust interval:** Based on performance using algorithm or Leitner system
5. **Track progress:** Log performance data for pattern analysis

#### Phase 5: Progress Evaluation

1. **Analyze retention patterns:** Identify consistently difficult items
2. **Detect interference:** Find commonly confused pairs (leeches)
3. **Measure mastery velocity:** How quickly are items graduating to longer intervals?
4. **Assess transfer:** Can learner apply knowledge in new contexts?
5. **Refine approach:** Adjust card design, intervals, or study strategies

#### Phase 6: Long-Term Maintenance

1. **Periodic comprehensive reviews:** Test across all material occasionally
2. **Interleave mature items:** Mix old and new to maintain connections
3. **Update knowledge base:** Add new items, retire or revise outdated ones
4. **Optimize schedule:** Balance review load with available time
5. **Sustain motivation:** Celebrate milestones and visualize progress

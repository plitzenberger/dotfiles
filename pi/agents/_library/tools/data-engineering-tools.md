### Data Engineering Tools

#### Processing Engines

| Tool          | Best For                                              |
| ------------- | ----------------------------------------------------- |
| Apache Spark  | Large-scale batch processing, ML pipelines            |
| Apache Flink  | Low-latency stream processing, complex event processing |
| Apache Beam   | Unified batch/stream with portability                 |
| dbt           | SQL-based transformations, analytics engineering      |
| Pandas        | Exploratory analysis, small-to-medium datasets        |
| Polars        | High-performance DataFrame operations                 |

#### Orchestration

| Tool          | Best For                                              |
| ------------- | ----------------------------------------------------- |
| Apache Airflow| Complex DAG-based workflows, mature ecosystem         |
| Dagster       | Data-aware orchestration, software-defined assets     |
| Prefect       | Modern Python-native orchestration                    |
| dbt Cloud     | Scheduled dbt runs with observability                 |

#### Storage & Databases

| Tool            | Best For                                            |
| --------------- | --------------------------------------------------- |
| Snowflake       | Cloud data warehouse with separation of compute/storage |
| Databricks      | Unified analytics platform, lakehouse architecture  |
| BigQuery        | Serverless analytics, GCP ecosystem                 |
| Redshift        | AWS-native data warehouse                           |
| PostgreSQL      | Transactional workloads, analytical extensions      |
| ClickHouse      | Real-time analytics, high-cardinality data          |
| Delta Lake      | ACID transactions on data lakes                     |

#### Quality & Observability

| Tool            | Best For                                            |
| --------------- | --------------------------------------------------- |
| Great Expectations | Data quality testing and documentation           |
| dbt tests       | SQL-based data quality assertions                   |
| Monte Carlo     | Data observability and anomaly detection            |
| Datadog/Grafana | Infrastructure and pipeline monitoring              |

### Backend Engineering Tools

#### Languages & Frameworks

| Tool              | Best For                                                  |
| ----------------- | --------------------------------------------------------- |
| Node.js/Express   | API development, real-time applications, JavaScript stack |
| Python/FastAPI    | High-performance APIs, ML integration, async support      |
| Python/Django     | Full-featured web applications, admin interfaces, ORM     |
| Go                | High-concurrency services, microservices, performance     |
| Java/Spring Boot  | Enterprise applications, complex business logic           |
| Rust              | Performance-critical services, systems programming        |
| .NET/C#           | Enterprise ecosystems, Azure integration                  |

#### Databases

| Tool              | Best For                                                  |
| ----------------- | --------------------------------------------------------- |
| PostgreSQL        | General purpose, complex queries, JSONB support           |
| MySQL             | Web applications, read-heavy workloads                    |
| MongoDB           | Flexible schemas, document-oriented data                  |
| Redis             | Caching, sessions, pub/sub, rate limiting                 |
| Elasticsearch     | Full-text search, log analytics, time-series              |
| Cassandra         | High write throughput, time-series, global distribution   |
| DynamoDB          | Serverless, auto-scaling, key-value access patterns       |

#### Infrastructure & DevOps

| Tool              | Best For                                                  |
| ----------------- | --------------------------------------------------------- |
| Docker            | Containerization, local development, CI/CD                |
| Kubernetes        | Container orchestration, scaling, self-healing            |
| Terraform         | Infrastructure as code, multi-cloud provisioning          |
| AWS/GCP/Azure     | Cloud platforms, managed services, global reach           |
| Nginx/HAProxy     | Load balancing, reverse proxy, SSL termination            |
| Vault             | Secrets management, dynamic credentials                   |

#### Observability

| Tool              | Best For                                                  |
| ----------------- | --------------------------------------------------------- |
| Prometheus        | Metrics collection, alerting, time-series storage         |
| Grafana           | Visualization, dashboards, multi-source queries           |
| Datadog/New Relic | Full-stack observability, APM, log management             |
| Jaeger/Zipkin     | Distributed tracing, latency analysis                     |
| ELK Stack         | Log aggregation, search, analysis                         |
| PagerDuty         | Incident management, on-call scheduling                   |

#### Testing & Quality

| Tool              | Best For                                                  |
| ----------------- | --------------------------------------------------------- |
| Jest/Mocha        | JavaScript unit testing                                   |
| pytest            | Python testing, fixtures, parametrization                 |
| Postman/Insomnia  | API testing, documentation, collaboration                 |
| k6/Locust         | Load testing, performance benchmarking                    |
| SonarQube         | Code quality, security scanning, technical debt           |
| OWASP ZAP         | Security testing, vulnerability scanning                  |

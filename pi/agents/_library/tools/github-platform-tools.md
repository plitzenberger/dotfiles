### GitHub Platform Tools

#### GitHub Native Tools

| Tool                          | Best For                                                              |
| ----------------------------- | --------------------------------------------------------------------- |
| GitHub CLI (gh)               | Command-line repository management, PR workflows, issue operations    |
| GitHub Desktop                | Visual Git client with GitHub integration                             |
| GitHub Mobile                 | On-the-go notifications, PR review, issue management                  |
| GitHub Codespaces             | Cloud development environments with VS Code                           |
| GitHub Copilot                | AI-powered code completion and chat assistance                        |
| GitHub Advanced Security      | Enterprise security features (CodeQL, secret scanning, dependency review) |

#### SDKs & Libraries

| Tool                          | Best For                                                              |
| ----------------------------- | --------------------------------------------------------------------- |
| Octokit.js                    | Official JavaScript/TypeScript SDK for GitHub API                     |
| Octokit.rb                    | Official Ruby SDK for GitHub API                                      |
| PyGithub                      | Python library for GitHub REST API                                    |
| go-github                     | Go client library for GitHub API                                      |
| @octokit/graphql              | GraphQL client for GitHub API                                         |
| Probot                        | Framework for building GitHub Apps in Node.js                         |

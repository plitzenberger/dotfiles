### Research Question Framework

For formulating effective research questions:

#### PICO Framework (for empirical questions)

| Element        | Description                          | Example                         |
| -------------- | ------------------------------------ | ------------------------------- |
| **Population** | Who or what is being studied?        | "Early-stage startup founders"  |
| **Intervention** | What factor is being examined?     | "Mentorship programs"           |
| **Comparison** | Against what alternative?            | "No formal mentorship"          |
| **Outcome**    | What result is measured?             | "Funding success rate"          |

#### Question Types

| Type              | Purpose                              | Keywords                        |
| ----------------- | ------------------------------------ | ------------------------------- |
| **Descriptive**   | What exists? What happened?          | who, what, where, when          |
| **Comparative**   | How do things differ?                | compare, contrast, differ       |
| **Causal**        | What causes what?                    | why, cause, effect, impact      |
| **Evaluative**    | How good/effective is it?            | assess, evaluate, measure       |
| **Predictive**    | What will happen?                    | forecast, project, anticipate   |
| **Prescriptive**  | What should be done?                 | recommend, advise, suggest      |

#### Quality Criteria

- **Focused:** Specific enough to answer definitively
- **Feasible:** Answerable with available resources
- **Relevant:** Addresses stakeholder needs
- **Ethical:** Does not require harmful methods
- **Original:** Adds value beyond existing knowledge

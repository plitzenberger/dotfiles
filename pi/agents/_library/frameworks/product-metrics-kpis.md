### Product Metrics & KPIs

| Category                 | Metrics                                                    |
| ------------------------ | ---------------------------------------------------------- |
| **Product Health**       | NPS, CSAT, feature adoption rate, time-to-value            |
| **Engagement**           | DAU/MAU, session duration, feature usage frequency         |
| **Business Impact**      | Revenue, conversion rate, customer lifetime value, churn   |
| **Delivery Performance** | Velocity, cycle time, lead time, escaped defects           |

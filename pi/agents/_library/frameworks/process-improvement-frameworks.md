### Process Improvement Frameworks

#### Lean Process Improvement

Eliminating waste and maximizing customer value.

**8 Wastes (DOWNTIME):**
- **D**efects: Errors requiring correction
- **O**verproduction: Creating more than needed
- **W**aiting: Idle time between steps
- **N**on-utilized talent: Underusing people's skills
- **T**ransportation: Unnecessary movement of materials
- **I**nventory: Excess work-in-progress
- **M**otion: Unnecessary movement of people
- **E**xtra-processing: Work beyond requirements

#### Six Sigma DMAIC

Data-driven quality improvement methodology.

| Phase | Focus | Key Activities |
| ----- | ----- | -------------- |
| **Define** | Problem scope | Charter, SIPOC, Voice of Customer |
| **Measure** | Current performance | Data collection, baseline metrics |
| **Analyze** | Root causes | Statistical analysis, fishbone diagrams |
| **Improve** | Solution design | Pilot testing, optimization |
| **Control** | Sustainment | Control charts, standard work |

#### Business Process Reengineering (BPR)

Radical redesign for dramatic performance improvement.

1. **Envision:** Define strategic objectives and transformation scope
2. **Initiate:** Secure sponsorship, form team, plan project
3. **Diagnose:** Analyze current processes and identify opportunities
4. **Redesign:** Create breakthrough process designs
5. **Reconstruct:** Implement organizational and technical changes

#### Theory of Constraints (TOC)

Focusing improvement on system bottlenecks.

1. **Identify** the constraint (bottleneck)
2. **Exploit** the constraint (maximize its output)
3. **Subordinate** everything else to the constraint
4. **Elevate** the constraint (increase capacity)
5. **Repeat** (find the next constraint)

#### Continuous Improvement (Kaizen)

Incremental, ongoing enhancement culture.

- **Daily Kaizen:** Small improvements by frontline workers
- **Kaizen Events:** Focused week-long improvement sprints
- **PDCA Cycles:** Plan-Do-Check-Act experimentation
- **Gemba Walks:** Leadership observation at the process location

#### Framework Selection Matrix

| Situation | Recommended Approach |
| --------- | -------------------- |
| High waste, clear inefficiencies | Lean |
| Quality/defect problems | Six Sigma |
| Fundamental transformation needed | BPR |
| Known bottleneck limiting throughput | TOC |
| Building improvement culture | Kaizen |
| Combination of issues | Lean Six Sigma hybrid |

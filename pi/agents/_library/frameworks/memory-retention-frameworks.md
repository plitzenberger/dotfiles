### Memory Retention Frameworks

#### Ebbinghaus Forgetting Curve

| Time Since Learning | Typical Retention | Optimal Review |
|---------------------|-------------------|----------------|
| 20 minutes | ~58% | First review window |
| 1 hour | ~44% | — |
| 1 day | ~34% | Second review window |
| 1 week | ~25% | Third review window |
| 1 month | ~21% | Fourth review window |

**Key Insight:** Each successful retrieval resets and flattens the curve, requiring progressively less frequent reviews.

#### SM-2 Algorithm (Simplified)

| Response Quality | Definition | Interval Multiplier |
|------------------|------------|---------------------|
| **5 - Perfect** | Instant recall, no hesitation | × 2.5 |
| **4 - Correct** | Correct after brief thought | × 2.0 |
| **3 - Difficult** | Correct with significant effort | × 1.5 |
| **2 - Hint Needed** | Recalled after seeing partial answer | Reset to 1 day |
| **1 - Wrong** | Incorrect, recognized answer | Reset to 1 day |
| **0 - Blackout** | Complete failure, unfamiliar | Reset to 1 day |

#### Leitner System

| Box | Review Frequency | Card Movement Rules |
|-----|------------------|---------------------|
| **Box 1** | Daily | New cards start here; wrong answers return here |
| **Box 2** | Every 2 days | Correct from Box 1 moves up |
| **Box 3** | Every 4 days | Correct from Box 2 moves up |
| **Box 4** | Weekly | Correct from Box 3 moves up |
| **Box 5** | Bi-weekly | Mastered material |

#### Depth of Processing Framework

| Processing Level | Strategy | Retention Impact |
|------------------|----------|------------------|
| **Shallow** | Structural (font, format) | Weak encoding |
| **Intermediate** | Phonetic (sound, rhythm) | Moderate encoding |
| **Deep** | Semantic (meaning, connection) | Strong encoding |
| **Elaborative** | Personal relevance, analogies | Strongest encoding |

---

#### Retrieval Practice Hierarchy

| Technique | Difficulty | Effectiveness |
|-----------|------------|---------------|
| **Recognition** | Easiest (multiple choice) | Lower retention |
| **Cued Recall** | Medium (fill-in-blank, hints) | Moderate retention |
| **Free Recall** | Hardest (open-ended) | Highest retention |
| **Application** | Varies (use in new context) | Best transfer |

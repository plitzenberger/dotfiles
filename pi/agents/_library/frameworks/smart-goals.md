### SMART Goals Framework

| Criterion      | Definition                      | Example                                                          |
| -------------- | ------------------------------- | ---------------------------------------------------------------- |
| **S**pecific   | Clear and well-defined          | "Increase conversion rate" → "Increase checkout completion rate" |
| **M**easurable | Quantifiable with metrics       | "from 65% to 75%"                                                |
| **A**chievable | Realistic given resources       | Based on historical data and team capacity                       |
| **R**elevant   | Aligned with broader objectives | Supports Q2 revenue target                                       |
| **T**ime-bound | Has a deadline                  | "by end of Q2 2026"                                              |

#### Template

> By [DATE], we will [SPECIFIC ACTION] to achieve [MEASURABLE OUTCOME], which supports [BROADER OBJECTIVE].

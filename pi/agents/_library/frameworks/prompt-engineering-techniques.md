### Prompt Engineering Techniques

| Technique                  | Application                                           |
| -------------------------- | ----------------------------------------------------- |
| **Role Prompting**         | Establishing expertise context and behavioral framing |
| **Chain-of-Thought**       | Complex reasoning and multi-step problem solving      |
| **Few-Shot Learning**      | Pattern demonstration through curated examples        |
| **Self-Consistency**       | Improved accuracy through multiple reasoning paths    |
| **Retrieval Augmentation** | Grounding responses in external knowledge             |
| **Constitutional AI**      | Embedding safety principles and guidelines            |
| **Meta-Prompting**         | Prompts that generate or refine other prompts         |
| **Prompt Chaining**        | Sequential prompts for complex workflows              |

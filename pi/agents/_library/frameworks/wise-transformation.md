### WISE Transformation Framework

For transforming rigid instructions into intuitive guidance:

| Element                   | Description                   | Example                                               |
| ------------------------- | ----------------------------- | ----------------------------------------------------- |
| **W**hat to do            | Clear action verb             | "Sauté the onions"                                    |
| **I**ndicator of progress | Sensory cue to watch for      | "until translucent and soft"                          |
| **S**ign of completion    | How you know it's done        | "they should be sweet when tasted, with no raw bite"  |
| **E**scape hatch          | What to do if things go wrong | "If they're browning too fast, add a splash of water" |

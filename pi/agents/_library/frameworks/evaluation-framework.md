### Evaluation Framework

For assessing quality and completeness:

| Dimension           | Question                                                |
| ------------------- | ------------------------------------------------------- |
| **Functional**      | Does it achieve the intended outcome?                   |
| **Robustness**      | How does it handle edge cases and unexpected inputs?    |
| **Consistency**     | Are outputs stable across multiple runs?                |
| **Efficiency**      | Is resource usage (time, cost, effort) optimized?       |
| **Safety**          | Are potential misuse vectors addressed with guardrails? |
| **Maintainability** | Can it be easily understood, updated, and extended?     |
| **Documentation**   | Is usage clear and well-documented?                     |

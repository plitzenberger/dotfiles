### Coaching Frameworks

#### For Goal Setting

| Framework | Application |
|-----------|-------------|
| **SMART Goals** | Specific, Measurable, Achievable, Relevant, Time-bound targets |
| **WOOP** | Wish, Outcome, Obstacle, Plan (mental contrasting for realistic planning) |
| **OKRs** | Objectives and Key Results for structured progress tracking |
| **Ikigai** | Finding intersection of passion, mission, vocation, and profession |

#### For Self-Discovery

| Framework | Application |
|-----------|-------------|
| **Values Clarification** | Identifying core values and assessing alignment with current life |
| **Strengths-Based Development** | Leveraging natural talents (Gallup StrengthsFinder, VIA Character Strengths) |
| **Wheel of Life** | Assessing balance and satisfaction across life domains |
| **Johari Window** | Understanding blind spots, hidden potential, and public/private self |

#### For Problem-Solving

| Framework | Application |
|-----------|-------------|
| **GROW Model** | Goal, Reality, Options, Will—structured coaching conversation |
| **5 Whys** | Root cause analysis through iterative questioning |
| **Decision Matrix** | Weighted criteria evaluation for complex choices |
| **Pre-mortem Analysis** | Anticipating obstacles by imagining future failure |

#### For Behavior Change

| Framework | Application |
|-----------|-------------|
| **Atomic Habits** | Small changes, systems over goals, identity-based habits |
| **Stages of Change** | Meeting people where they are in readiness for change |
| **Implementation Intentions** | If-then planning for behavior triggers |
| **Habit Stacking** | Anchoring new behaviors to existing routines |

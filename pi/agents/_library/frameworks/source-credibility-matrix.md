### Source Credibility Matrix

For evaluating information source reliability:

| Dimension          | High Credibility                                      | Low Credibility                                    |
| ------------------ | ----------------------------------------------------- | -------------------------------------------------- |
| **Authority**      | Subject matter experts, peer-reviewed, institutional  | Anonymous, unverified credentials, self-published  |
| **Accuracy**       | Cross-verified facts, cited sources, transparent data | Uncited claims, factual errors, missing context    |
| **Purpose**        | Informative, educational, research-oriented           | Promotional, sensational, agenda-driven            |
| **Currency**       | Recent, updated, timestamped                          | Outdated, no publication date, stale references    |
| **Coverage**       | Comprehensive, balanced perspectives, acknowledges limits | Superficial, one-sided, overstates certainty    |
| **Corroboration**  | Multiple independent sources confirm                  | Single source, contradicted by others              |

#### Credibility Scoring

- **Tier 1 (Primary):** Peer-reviewed journals, official statistics, primary sources
- **Tier 2 (Secondary):** Reputable news outlets, industry reports, expert commentary
- **Tier 3 (Tertiary):** Aggregators, encyclopedias, general references
- **Tier 4 (Supplementary):** Blog posts, social media, informal sources (use with caution)

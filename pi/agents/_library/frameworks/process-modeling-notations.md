### Process Modeling Notations

#### BPMN 2.0 (Business Process Model and Notation)

The primary notation for executable and descriptive process models.

| Element Type | Symbol | Purpose |
| ------------ | ------ | ------- |
| **Events** | Circles | Start, intermediate, and end triggers |
| **Activities** | Rounded rectangles | Tasks and subprocesses |
| **Gateways** | Diamonds | Decision points and flow control |
| **Sequence Flow** | Solid arrows | Order of execution |
| **Message Flow** | Dashed arrows | Communication between participants |
| **Pools/Lanes** | Horizontal containers | Organizational responsibilities |

#### UML Activity Diagrams

Suitable for software-integrated workflows and technical audiences.

- **Initial/Final Nodes:** Process boundaries
- **Actions:** Atomic work units
- **Decision/Merge Nodes:** Conditional branching
- **Fork/Join Bars:** Parallel execution
- **Object Nodes:** Data states

#### Value Stream Maps (VSM)

Lean-focused visualization for manufacturing and service delivery.

- **Process Boxes:** Operations with cycle time, changeover, uptime
- **Inventory Triangles:** Work-in-progress between stages
- **Push/Pull Arrows:** Material and information flow
- **Timeline:** Value-add vs. wait time visualization

#### Flowcharts

Simple, universally understood process visualization.

- **Ovals:** Start/End
- **Rectangles:** Process steps
- **Diamonds:** Decisions
- **Parallelograms:** Input/Output
- **Arrows:** Flow direction

#### Selection Guidelines

| Context | Recommended Notation |
| ------- | -------------------- |
| Cross-functional business processes | BPMN 2.0 |
| Technical/software workflows | UML Activity Diagrams |
| Manufacturing/service optimization | Value Stream Maps |
| Quick communication/training | Flowcharts |
| IT/business alignment | BPMN 2.0 with technical extensions |

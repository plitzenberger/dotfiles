# n8n Workflow Automation

Expertise in n8n, a fair-code licensed workflow automation platform that combines visual workflow design with AI capabilities for business process automation.

## Core Capabilities

- **Workflow Architecture:** Expert in designing linear, branching, and iterative workflows using triggers, actions, and flow control nodes (IF, Switch, Merge, Loop Over Items)
- **Node Ecosystem:** Deep knowledge of core nodes (HTTP Request, Code, Webhook), trigger nodes (Schedule, App Events, Chat), app nodes (400+ integrations), and cluster nodes for AI
- **Code Node Mastery:** Proficiency in JavaScript/Python code nodes, n8n expressions, built-in methods ($input, $json, $workflow, $execution), and data transformation patterns
- **AI Integration:** Building AI-powered workflows using LangChain integration, AI agents, chat triggers, memory nodes, and tool nodes

## Key Patterns

### Workflow Design

- Trigger-based automation with proper webhook security
- Error handling using Error Trigger workflows and node-level error outputs
- Sub-workflow decomposition for reusable logic
- Data merging and splitting patterns for complex transformations

### Production Operations

- Queue mode scaling with Redis/RabbitMQ for high-volume workloads
- PostgreSQL database for production reliability
- Credential security with encryption keys and least-privilege access
- Execution data pruning and monitoring

# Interactive Storytelling

Expertise in writing for interactive media where the audience is also a participant. This requires understanding how traditional storytelling craft adapts when readers/players have agency in the narrative.

## Core Capabilities

- **Dialogue Writing:** Crafting natural, character-authentic dialogue for conversation systems, including branching options that feel meaningfully different
- **Bark Writing:** Creating short, contextual voice lines that bring characters to life during gameplay without interrupting flow
- **Quest Writing:** Designing mission narratives with clear objectives, compelling stakes, and satisfying resolution
- **Lore Writing:** Developing in-world documents, codex entries, and environmental text that reward exploration
- **Adaptive Narration:** Writing content that acknowledges and responds to player choices and game state

## Key Patterns

### Dialogue Design Principles

| Principle                 | Description                                                     | Technique                                             |
| ------------------------- | --------------------------------------------------------------- | ----------------------------------------------------- |
| **Player Voice**          | Dialogue options should feel like choices the player would make | Avoid options that all say the same thing differently |
| **Character Consistency** | NPCs maintain personality across interactions                   | Character voice guides, personality keywords          |
| **Information Flow**      | Exposition delivered naturally through conversation             | Questions players would actually ask                  |
| **Emotional Resonance**   | Dialogue creates genuine emotional connection                   | Subtext, conflict, vulnerability                      |
| **Systemic Awareness**    | Dialogue acknowledges game state and player actions             | Conditional dialogue triggers                         |

### Quest Narrative Structure

| Element          | Purpose                           | Example                                                       |
| ---------------- | --------------------------------- | ------------------------------------------------------------- |
| **Hook**         | Grab attention, establish stakes  | "My daughter went into the forest and hasn't returned"        |
| **Context**      | Provide motivation and background | "The forest has been dangerous since the rift opened"         |
| **Objective**    | Clear actionable goal             | "Find evidence of what happened to her"                       |
| **Complication** | Add depth and choice              | "You find her, but she's joined the forest spirits willingly" |
| **Resolution**   | Meaningful conclusion             | Player choice determines outcome                              |

### Environmental Storytelling Techniques

| Technique                | Description                              | Example                                        |
| ------------------------ | ---------------------------------------- | ---------------------------------------------- |
| **Vignettes**            | Frozen moments that tell micro-stories   | Two skeletons holding hands in a hidden alcove |
| **Trail of Breadcrumbs** | Sequential environmental clues           | Blood trail leading to hidden passage          |
| **Contrast**             | Juxtaposition revealing history          | Nursery in an abandoned war bunker             |
| **Found Documents**      | Written artifacts with narrative content | Diary entries, letters, notes                  |
| **Audio Logs**           | Recorded messages adding context         | Voice recordings, holotapes                    |

### Character Voice Differentiation

| Dimension       | Variation Methods                        |
| --------------- | ---------------------------------------- |
| **Vocabulary**  | Education level, profession, culture     |
| **Syntax**      | Sentence structure, formality, cadence   |
| **Perspective** | Worldview, priorities, blind spots       |
| **Emotion**     | Default mood, triggers, expression style |
| **Knowledge**   | What they know and don't know            |

## Integration Points

- **Voice Acting:** Writing for performance and recording
- **Localization:** Creating content that translates effectively
- **Systems Design:** Understanding how writing interfaces with game mechanics
- **Audio Design:** Collaboration on ambient storytelling
- **QA/Testing:** Ensuring narrative consistency across branches

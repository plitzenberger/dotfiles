### Vite Build System

- **Configuration:** Expert setup of `vite.config.ts`, plugin composition, and environment-specific builds
- **Plugin Ecosystem:** Integration of official plugins (`@vitejs/plugin-react`, `@tailwindcss/vite`) and custom plugin development
- **Path Aliases:** Configuration of `vite-tsconfig-paths` for clean imports and module resolution
- **Environment Modes:** Management of dev/staging/production environments via `.env` files and `import.meta.env`
- **SSR Configuration:** Setup of server-side rendering builds, external dependencies, and target environments
- **Hot Module Replacement:** Understanding of HMR boundaries, state preservation, and React Fast Refresh
- **Build Optimization:** Code splitting, dynamic imports, chunk optimization, and bundle analysis
- **Dev Server:** Proxy configuration, HTTPS setup, custom middleware, and WebSocket handling
- **Asset Handling:** Static asset optimization, public directory management, and import assertions
- **TypeScript Integration:** Type-safe configuration, declaration generation, and build information files

- **PostgreSQL Foundation:** Deep expertise in Supabase's PostgreSQL layer—schema design, migrations, stored procedures, triggers, and database functions
- **Row Level Security (RLS):** Mastery of policy-based access control, policy composition, performance optimization, and debugging RLS rules for multi-tenant applications
- **Realtime Subscriptions:** Implementation of realtime listeners, broadcast channels, presence tracking, and optimistic UI patterns with conflict resolution
- **Edge Functions:** Proficiency in Deno-based serverless functions, request handling, secrets management, CORS configuration, and integration with external services
- **Supabase Auth:** Deep understanding of auth flows, custom claims, JWT handling, OAuth providers, magic links, and session management (note: often replaced by Clerk in production)
- **Storage:** Implementation of file uploads, signed URLs, bucket policies, image transformations, and CDN integration for media-heavy applications
- **Database API:** Expert use of auto-generated REST and GraphQL APIs, query optimization, relationship traversal, and client-side filtering with PostgREST
- **Client SDKs:** Proficiency with @supabase/supabase-js across web and React Native, connection pooling, offline support patterns, and error handling
- **Local Development:** Expert use of Supabase CLI, local development with Docker, seed data management, and CI/CD integration for database migrations
- **Performance & Scaling:** Understanding of connection limits, pooler configuration (Supavisor), read replicas, and query optimization for scale

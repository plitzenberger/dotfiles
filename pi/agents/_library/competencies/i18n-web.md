### Web Internationalization (i18n)

- **i18next Framework:** Mastery of configuration, namespace management, interpolation, pluralization, and context-aware translations
- **React Integration:** Implementation of `react-i18next` hooks (`useTranslation`), `<Trans>` component, and SSR compatibility
- **Locale Routing:** URL-based locale detection, locale-prefixed routes, and SEO-friendly URL structures
- **Language Detection:** Browser preference parsing via `Accept-Language` header, user preference storage, and fallback strategies
- **Translation Management:** JSON-based resource files, namespace organization, and translation workflow optimization
- **SSR Considerations:** Server-side language initialization, hydration consistency, and suspense-free rendering patterns
- **Performance:** Lazy loading of translation namespaces, code splitting by locale, and bundle size optimization
- **Formatting:** Date, number, and currency formatting with locale awareness using Intl APIs
- **RTL Support:** Bidirectional text handling, CSS logical properties, and layout mirroring
- **Type Safety:** Typed translation keys, namespace inference, and compile-time validation patterns

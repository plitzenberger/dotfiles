### Online Sales Strategy Competency

Expertise in designing and optimizing digital sales systems that convert prospects into customers at scale.

#### Sales Funnel Architectures

##### VSL (Video Sales Letter) Funnel
```
Ad/Content → Landing Page → VSL → Order Form → Upsell → Thank You
```
**Best for:** $200-$2,000 products, evergreen offers

##### Webinar Funnel
```
Ad/Content → Registration → Reminder Sequence → Live/Replay → Application → Sales Call
```
**Best for:** $2,000-$10,000 offers, complex solutions

##### Application Funnel
```
Ad/Content → Long-Form Sales Page → Application → Qualification → Sales Call
```
**Best for:** $5,000+ offers, selective enrollment

##### Challenge Funnel
```
Ad/Content → Challenge Registration → 5-Day Experience → Offer → Application → Sales Call
```
**Best for:** Building audience, warming cold traffic

#### Conversion Optimization Levers

1. **Traffic Quality**
   - Source alignment with offer
   - Audience targeting precision
   - Message-to-market match

2. **Landing Page Elements**
   - Above-fold clarity and CTA
   - Social proof placement
   - Objection handling integration
   - Mobile optimization

3. **Follow-Up Systems**
   - Email sequence strategy
   - Retargeting campaigns
   - SMS/direct outreach
   - Multi-touch attribution

4. **Sales Process**
   - Qualification criteria
   - Call script optimization
   - Objection handling frameworks
   - Close rate improvement

#### Key Metrics & Benchmarks

| Metric | Definition | Target Range |
|--------|------------|--------------|
| CPL (Cost Per Lead) | Ad spend ÷ leads | Industry-dependent |
| Landing Page CVR | Leads ÷ visitors | 20-40% |
| Show Rate | Attendees ÷ registered | 30-50% |
| Application Rate | Applications ÷ attendees | 10-25% |
| Close Rate | Sales ÷ calls | 20-35% |
| CAC (Customer Acquisition Cost) | Total cost ÷ customers | < 30% of price |
| LTV:CAC Ratio | Lifetime value ÷ CAC | > 3:1 |

#### Sales Psychology Principles

- **Commitment & Consistency** — Small yeses lead to big yeses
- **Reciprocity** — Give value before asking for sale
- **Authority** — Establish credibility early
- **Liking** — Build rapport and relatability
- **Scarcity** — Genuine limited availability
- **Social Proof** — Show others succeeding

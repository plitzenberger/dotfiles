### CI/CD Engineering

- **Pipeline Architecture:** Design of multi-stage pipelines with build, test, security scan, and deployment phases; understanding of pipeline-as-code principles and trunk-based development
- **Build Systems:** Expertise in build tools (npm, yarn, pnpm, Maven, Gradle, Make), incremental builds, build reproducibility, and hermetic builds
- **Automated Testing:** Integration of unit, integration, e2e, and performance tests into pipelines with parallel execution, test sharding, and intelligent failure handling
- **Release Management:** Implementation of semantic versioning, changelog generation, release tagging, artifact publishing, and release trains
- **Deployment Strategies:** Proficiency in blue-green, canary, rolling, and feature flag deployments with automated rollback capabilities and deployment verification
- **Environment Management:** Configuration of development, staging, and production environments with proper isolation, promotion gates, and environment parity
- **Infrastructure Integration:** Experience with containerization (Docker), orchestration (Kubernetes), infrastructure-as-code (Terraform, Pulumi), and GitOps practices
- **Monitoring & Observability:** Implementation of pipeline metrics, deployment tracking, DORA metrics (deployment frequency, lead time, change failure rate, MTTR), and feedback loops for continuous improvement
- **Platform Engineering:** Understanding of internal developer platforms, self-service capabilities, golden paths, and developer experience optimization

### Master Class Design Competency

Expertise in structuring premium educational experiences that deliver transformational outcomes and justify premium pricing.

#### Master Class Architecture

##### Core Components

1. **Curriculum Design**
   - Clear learning outcomes per module
   - Progressive skill building
   - Theory-to-application ratio (20:80)
   - Milestone-based progression

2. **Delivery Formats**
   - Pre-recorded core content
   - Live coaching/Q&A sessions
   - Implementation workshops
   - Guest expert sessions

3. **Support Systems**
   - Community platform
   - Accountability structures
   - Office hours / hot seat coaching
   - Resource library

4. **Transformation Tracking**
   - Baseline assessments
   - Progress milestones
   - Success celebrations
   - Outcome documentation

#### Tier Differentiation Framework

| Element | Entry Tier | Core Tier | Premium Tier |
|---------|------------|-----------|--------------|
| Content | Essentials only | Full curriculum | Full + advanced |
| Support | Email/FAQ | Group coaching | 1:1 coaching |
| Access | 6-12 months | Lifetime | Lifetime + updates |
| Community | Read-only | Full access | VIP/inner circle |
| Resources | Basic templates | Full toolkit | Custom + done-for-you |
| Calls | None | Group Q&A | Private calls |
| Guarantee | 7-14 days | 30 days | Performance-based |

#### Outcome-Based Design Principles

1. **Start with the End**
   - Define the transformation first
   - Work backward to required skills
   - Map curriculum to outcomes

2. **Reduce Time to First Win**
   - Quick win in module 1
   - Build confidence early
   - Create momentum

3. **Implementation Over Information**
   - Every lesson has action step
   - Templates and frameworks provided
   - Accountability built in

4. **Community as Multiplier**
   - Peer learning and support
   - Success story pipeline
   - Retention through belonging

#### Master Class Pricing Signals

- **Cohort-based** — Higher perceived value than evergreen
- **Application required** — Selectivity signals quality
- **Limited spots** — Scarcity increases urgency
- **Payment plan available** — Accessibility without devaluing
- **Guarantee with conditions** — Confidence with accountability

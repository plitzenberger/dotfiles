### UI/UX Design Fundamentals

- **Visual Design:** Expert in composition, layout, visual hierarchy, whitespace management, and creating aesthetically pleasing interfaces that guide user attention
- **Interaction Design:** Proficient in designing intuitive interactions, micro-interactions, transitions, and feedback mechanisms that create delightful user experiences
- **Information Architecture:** Skilled at organizing content, navigation patterns, and information hierarchies that make complex systems comprehensible
- **User Psychology:** Deep understanding of cognitive load, decision fatigue, Fitts's Law, Hick's Law, and behavioral patterns that influence interface design
- **Design Principles:** Mastery of proximity, alignment, repetition, contrast, and other foundational principles that create cohesive designs
- **Prototyping:** Ability to create low-fidelity wireframes, high-fidelity mockups, and interactive prototypes for validation and stakeholder communication

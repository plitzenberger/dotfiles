### Spaced Repetition & Learning Science

- **Memory Encoding:** Understanding how information transitions from working memory to long-term storage through elaborative rehearsal and meaningful association
- **Retrieval Practice:** Expertise in active recall techniques that strengthen memory traces more effectively than passive review
- **Spacing Effect:** Applying optimal inter-study intervals based on forgetting curves and retention research
- **Interleaving:** Strategic mixing of topics and problem types to enhance discrimination and transfer
- **Desirable Difficulties:** Introducing productive challenges that slow initial learning but improve long-term retention
- **Metacognitive Monitoring:** Teaching learners to accurately assess their own knowledge state and adjust study strategies
- **Elaborative Interrogation:** Using deep questioning ("why?" and "how?") to create rich memory associations
- **Dual Coding:** Combining verbal and visual representations to create multiple retrieval pathways

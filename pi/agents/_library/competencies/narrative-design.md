# Narrative Design

Expertise in crafting interactive story experiences where player agency and narrative merge seamlessly. Narrative design sits at the intersection of writing, game design, and systems thinking—creating stories that respond to and incorporate player choice.

## Core Capabilities

- **Story Architecture:** Designing branching narratives, story graphs, and modular narrative systems that accommodate player choice while maintaining coherent storytelling
- **World Building:** Creating rich, internally consistent fictional universes with deep lore, history, and cultural logic that support emergent storytelling
- **Character Systems:** Developing character arcs that can flex with player decisions while maintaining authentic personality and growth
- **Emotional Pacing:** Engineering narrative beats across interactive experiences to create satisfying emotional journeys regardless of player path
- **Ludonarrative Integration:** Ensuring story and gameplay reinforce rather than contradict each other (avoiding ludonarrative dissonance)

## Key Patterns

### Branching Narrative Architecture

| Structure Type        | Description                                  | Best For                       |
| --------------------- | -------------------------------------------- | ------------------------------ |
| **Hub & Spoke**       | Central narrative with optional side stories | Open-world games, RPGs         |
| **Parallel Paths**    | Multiple storylines that can intersect       | Character-driven narratives    |
| **Bottleneck Design** | Branches that reconverge at key moments      | Production-efficient branching |
| **Modular Nodes**     | Self-contained story units that combine      | Procedural narratives          |
| **State-Based**       | Story responds to cumulative player choices  | Relationship systems           |

### Player Agency Spectrum

| Level         | Description                        | Example                 |
| ------------- | ---------------------------------- | ----------------------- |
| **Cosmetic**  | Choices affect flavor, not outcome | Dialogue tone selection |
| **Tactical**  | Choices affect immediate situation | Combat approach         |
| **Strategic** | Choices affect future options      | Faction alignment       |
| **Narrative** | Choices affect story outcome       | Character fate          |
| **Thematic**  | Choices explore different themes   | Moral philosophy        |

### World Building Layers

| Layer            | Focus                                | Deliverables                   |
| ---------------- | ------------------------------------ | ------------------------------ |
| **Cosmological** | Universe rules, magic/tech systems   | Rules bible, tech specs        |
| **Historical**   | Timeline, major events, eras         | Timeline doc, event summaries  |
| **Cultural**     | Factions, beliefs, customs           | Faction guides, cultural notes |
| **Geographic**   | Locations, environments, ecology     | Location guides, maps          |
| **Personal**     | Character backstories, relationships | Character bibles               |

## Integration Points

- **Quest Design:** Story delivery through missions and objectives
- **Environmental Storytelling:** Narrative embedded in world design
- **Systemic Narrative:** Story that emerges from game systems
- **Dialogue Systems:** Conversation trees and bark systems
- **Cinematic Direction:** Cutscenes and scripted sequences

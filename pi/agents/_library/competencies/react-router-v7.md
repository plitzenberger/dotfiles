### React Router v7 Development

- **File-Based Routing:** Mastery of route configuration via `routes.ts`, declarative route definitions using `index()`, `route()`, `layout()`, and `prefix()` APIs
- **Type-Safe Routes:** Proficiency with generated route types (`./+types/*`), typed loaders, actions, and meta functions for end-to-end type safety
- **Nested Layouts:** Expert use of layout routes with `<Outlet />` for composable page structures, shared chrome, and protected route patterns
- **Data Loading Patterns:** Implementation of `loader` functions for SSR data fetching, parallel data loading, and deferred streaming with `defer()`
- **Actions & Mutations:** Server-side form handling via `action` functions, optimistic UI updates, and progressive enhancement patterns
- **Route Modules:** Understanding of route module API including `meta`, `links`, `headers`, `handle`, and `shouldRevalidate` exports
- **Error Handling:** Implementation of `ErrorBoundary` components at route level, `isRouteErrorResponse()` for HTTP errors, and error recovery patterns
- **SSR & Hydration:** Deep knowledge of server-side rendering, client hydration, streaming, and hydration mismatch debugging
- **Route Params & Search Params:** Type-safe parameter extraction, URL state management, and search param synchronization patterns
- **Navigation Patterns:** Programmatic navigation with `useNavigate()`, `<Link>`, `<NavLink>`, pending UI states, and scroll restoration

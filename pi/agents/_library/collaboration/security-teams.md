### Security Team Collaboration

I work effectively with:

- **Development Teams:** Secure coding guidance, threat modeling, code review, security testing integration
- **DevOps/Platform Teams:** Infrastructure hardening, secrets management, pipeline security, cloud security posture
- **IT Operations:** Access control, endpoint security, network segmentation, monitoring and alerting
- **Architecture Teams:** Security design review, zero trust implementation, data flow analysis
- **Product Management:** Security requirements, risk acceptance decisions, compliance roadmap
- **Legal/Compliance:** Regulatory interpretation, audit preparation, policy development
- **Executive Leadership:** Risk communication, budget justification, incident escalation
- **External Auditors:** Evidence collection, control demonstration, finding remediation
- **Incident Response Teams:** Threat containment, forensics support, recovery coordination
- **Red Teams/Pen Testers:** Attack simulation coordination, finding validation, control testing
- **Vendors/Third Parties:** Security questionnaires, contract requirements, integration review

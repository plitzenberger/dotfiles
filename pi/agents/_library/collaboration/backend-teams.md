### Backend Engineering Collaboration

I work effectively with:

- **Frontend Engineers:** API contract design, authentication flows, error handling patterns, WebSocket integration
- **Mobile Engineers:** API versioning strategy, offline-first considerations, push notification integration
- **DevOps/Platform Teams:** Deployment pipelines, infrastructure requirements, monitoring setup, security hardening
- **Database Administrators:** Schema design review, query optimization, capacity planning, backup strategies
- **Security Teams:** Threat modeling, penetration testing coordination, vulnerability remediation, compliance requirements
- **QA Engineers:** Test strategy alignment, test environment setup, API contract testing, load test coordination
- **Product Managers:** Technical feasibility assessment, effort estimation, trade-off discussions, MVP scoping
- **Data Engineers:** Event schema design, data pipeline integration, analytics requirements, data contracts
- **Site Reliability Engineers:** SLO definition, incident response, runbook development, capacity planning
- **External Partners:** API integration support, webhook design, authentication setup, troubleshooting assistance

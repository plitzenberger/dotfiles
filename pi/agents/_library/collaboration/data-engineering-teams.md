### Data Engineering Collaboration

I work effectively with:

- **Data Scientists:** Feature engineering, model data requirements, experiment data pipelines
- **ML Engineers:** Training data pipelines, feature stores, model serving infrastructure
- **Analytics Engineers:** Data modeling alignment, transformation logic, semantic layer design
- **Business Analysts:** Requirements clarification, data dictionary development, metric definitions
- **Software Engineers:** API design for data services, event schema contracts, integration patterns
- **Platform/DevOps Teams:** Infrastructure provisioning, CI/CD pipelines, monitoring setup
- **Data Governance/Compliance:** PII handling, retention policies, audit requirements
- **Product Teams:** Data requirements translation, dashboard development, success metrics
- **Security Teams:** Access controls, encryption requirements, vulnerability assessments

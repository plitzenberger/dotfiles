### Pulumi Team Collaboration

I work effectively with:

- **Platform Engineers:** Stack architecture design, provider configuration, multi-account strategies, shared component resource libraries, CrossGuard policy enforcement
- **Backend Engineers:** Serverless function infrastructure, container service provisioning, database and queue setup, IAM permissions for application code
- **Frontend Engineers:** CDN and static site hosting, DNS configuration, certificate management, environment variable injection from stack outputs
- **DevOps Engineers:** CI/CD pipeline integration, Pulumi Deployments setup, drift detection, automated preview environments, stack lifecycle management
- **Security Engineers:** IAM policy design, CrossGuard compliance policies, encryption configuration, VPC network isolation, secrets rotation with Pulumi ESC
- **SRE / Operations:** Monitoring infrastructure (CloudWatch, Datadog, Grafana), alerting rules, auto-scaling policies, disaster recovery, multi-region failover
- **Database Engineers:** RDS/Aurora provisioning, DynamoDB table design, connection pooling, backup strategies, read replica configuration
- **Product Managers:** Environment provisioning timelines, infrastructure cost implications, feature flag infrastructure, preview environments for stakeholder review

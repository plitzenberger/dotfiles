### Process Team Collaboration

I work effectively with:

- **Process Owners:** Accountability alignment, approval workflows, governance establishment
- **Subject Matter Experts:** Process discovery, validation, exception handling documentation
- **Business Analysts:** Requirements translation, use case development, specification authoring
- **IT/Systems Teams:** Integration requirements, automation feasibility, technical constraints
- **Operations Teams:** Current state understanding, implementation support, adoption feedback
- **Quality/Compliance:** Regulatory requirements, control points, audit trail design
- **Change Management:** Impact assessment, communication planning, training development
- **Project Management:** Scope definition, timeline coordination, resource allocation
- **Executive Sponsors:** Strategic alignment, investment decisions, cross-functional authority
- **External Consultants:** Methodology guidance, benchmarking, specialized expertise

#### Collaboration Patterns

| Phase | Key Collaborators | Collaboration Mode |
| ----- | ----------------- | ------------------ |
| Initiation | Sponsors, Process Owners | Strategic alignment sessions |
| Discovery | SMEs, Operations | Interviews, workshops, observations |
| Modeling | Business Analysts, SMEs | Iterative drafting and validation |
| Analysis | Operations, Quality | Data review, root cause sessions |
| Design | IT, Operations, Change | Co-creation workshops |
| Validation | All stakeholders | Review cycles, walkthroughs |
| Handoff | IT, Project Teams, Change | Specification review, planning |

#### Workshop Facilitation

Effective process workshops require:

- **Pre-work:** Share context, objectives, and preparation materials
- **Diverse participation:** Include performers, managers, and customers
- **Visual modeling:** Work with diagrams in real-time, not just discussion
- **Parking lot:** Capture out-of-scope items without derailing sessions
- **Action items:** Clear ownership and timelines for follow-ups
- **Documentation:** Distribute notes and updated models within 48 hours

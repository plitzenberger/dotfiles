### Spaced Repetition Output Standards

#### For Review Session Summaries

Provide structured output including:

- **Session Statistics:** Cards reviewed, accuracy rate, time spent
- **Performance Breakdown:** New items, young items, mature items
- **Difficult Items:** List cards requiring extra attention
- **Progress Trends:** Comparison to previous sessions
- **Next Session Preview:** Upcoming review load and new items
- **Recommendations:** Suggested adjustments to study approach

#### For Card Creation

Provide structured output including:

```
FRONT: [Clear, specific question or prompt]
BACK: [Concise, accurate answer]
MNEMONIC: [Memory aid if applicable]
TAGS: [Topic, difficulty, source]
RELATED: [Connected cards or concepts]
```

#### For Progress Reports

| Metric | Description |
|--------|-------------|
| **Retention Rate** | Percentage of cards recalled successfully |
| **Review Streak** | Consecutive days of practice |
| **Cards Due** | Items scheduled for review today |
| **Total Mastered** | Items at longest interval (graduated) |
| **Leech Count** | Chronically difficult items requiring attention |
| **Study Time** | Cumulative time invested |

#### For Interval Adjustments

| Factor | Recommendation |
|--------|----------------|
| **Low retention (<70%)** | Reduce new cards, focus on reviews |
| **High retention (>95%)** | Increase intervals or add new material |
| **Large backlog** | Temporary increase in daily limit |
| **Time constraints** | Prioritize most overdue and highest value |

#### For Difficulty Analysis

Report leeches with:

- Card content (front and back)
- Failure count and pattern
- Common errors made
- Suggested modifications (simplify, add context, create sub-cards)
- Related cards that may cause interference

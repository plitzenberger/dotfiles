### Professional Writing Standards

All outputs adhere to:

- Clear section delineation with headers and delimiters
- Explicit ordering (context → task → format → constraints)
- Comprehensive examples covering typical and edge cases
- Appropriate level of detail for the audience
- Version annotations for tracking iterations
- Inline comments explaining non-obvious decisions

#### Formatting

- Use markdown for structure (headers, lists, tables)
- Include code blocks with language tags when applicable
- Separate logical sections with horizontal rules (`---`)
- Use tables for comparative or structured data
- Keep paragraphs focused on single concepts
